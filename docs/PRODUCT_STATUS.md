## 🎯 製品概要

**LP Design Guideline Generator** は、LPのURLを入力するだけで、競合調査に基づいた論理的なデザインガイドラインを自動生成するAIツールです。

## 📊 技術的な制約・既知の問題

| 項目 | 状況 | 備考 |
|------|------|------|
| 生成時間 | 約1-3分 | AIエージェントの分析ステップ数に依存 |
| API制限 | 各APIの制限に依存 | Claude, Serper, Unsplash等 |
| スクレイピング | 一部サイトで失敗 | robots.txtやJavaScript依存サイト |
| PDF日本語 | 部分的対応 | 一部フォントで文字化けの可能性 |
| Supabase未設定時 | ローカルのみ | クラウド機能無効 |

## 🏗️ アーキテクチャ概要

```
┌─────────────────────────────────────────────────────────────┐
│                      フロントエンド                          │
│  Next.js 16 (App Router) + React 19 + Tailwind CSS          │
├─────────────────────────────────────────────────────────────┤
│                        API Layer                             │
│  ├─ /api/generate (従来モード)                              │
│  ├─ /api/generate/stream (SSEストリーミング)                │
│  └─ /api/guidelines/[id] (共有取得)                         │
├─────────────────────────────────────────────────────────────┤
│                      AIエージェント                          │
│  ReActパターン ─ 7ツール ─ 最大12ステップ                   │
├─────────────────────────────────────────────────────────────┤
│                    ガイドライン生成                          │
│  Layer1 → Layer2 → Layer3 → 参考実例 → 一貫性チェック       │
├─────────────────────────────────────────────────────────────┤
│                       データ層                               │
│  ローカルストレージ + Supabase (PostgreSQL)                 │
└─────────────────────────────────────────────────────────────┘
```
