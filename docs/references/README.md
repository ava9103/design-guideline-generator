# References

このフォルダには、デザインガイドライン生成に関するリファレンス資料を保存します。

## 用途

- デザインの参考資料
- 競合サイトのスクリーンショット
- UI/UXパターンの例
- カラーパレットやタイポグラフィの参考
- その他の参考文献やリンク集

## フォルダ構成（例）

```
references/
├── screenshots/      # スクリーンショット
├── color-palettes/   # カラーパレット参考
├── typography/       # タイポグラフィ参考
├── ui-patterns/      # UIパターン
└── competitors/      # 競合分析資料
```
